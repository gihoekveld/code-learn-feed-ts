# Feed web

This is a simple web application that allows you to create a feed of your favorite websites.

## Technologies used ☕️ 🐍 ⚛️

- ReactJS
- Typescript

## Features 📋

- View the feed
- Add a new comment
- Delete a comment
- Applaud a comment

## Installation 💻

Clone the repository and install the dependencies.

```bash
git clone
cd feed-web
npm install
```

## Usage 🚀

```bash
npm run dev
```
